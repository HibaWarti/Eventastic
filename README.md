# Eventastic

Eventastic is a social platform for event inspiration and service discovery.

Users can:
- Share events
- Discover ideas
- Connect with service providers
- Chat with professionals

## Architecture

The project uses:

- MERN stack
- Microservices architecture
- Docker containerization

## Services

- API Gateway
- Auth Service
- User Service
- Event Service
- Provider Service
- Chat Service
- Media Service

## Frontend

React + TailwindCSS

## Database

MongoDB
